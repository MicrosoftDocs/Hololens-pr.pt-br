# <a name="hololens-2-development-edition-faq"></a>Perguntas frequentes do HoloLens 2 Development Edition

## <a name="general-questions"></a>Perguntas gerais

### <a name="what-do-i-get-with-the-hololens-2-development-edition"></a>O que obtenho com a edição de desenvolvimento do HoloLens 2?

A edição de desenvolvimento do HoloLens 2 inclui um HoloLens 2, $500 em créditos do Azure, uma avaliação do Unity pro de 3 meses e uma avaliação de plugin de Pixyz de 3 meses.

### <a name="what-is-the-difference-between-hololens-2-and-hololens-2-development-edition"></a>Qual é a diferença entre o HoloLens 2 e o HoloLens 2 Development Edition?

Você só pode usar a edição de desenvolvimento do HoloLens 2 para criar e testar aplicativos empresariais internos a serem usados com o HoloLens 2 (edição comercial) em uma configuração de produção. Como parte da compra da edição de desenvolvimento, a Microsoft pode, em seu critério exclusivo, fornecer determinados direitos identificados durante o processo de check-out. Esses direitos incluem o direito de resgatar determinados créditos de desconto ou acesso de avaliação a determinados serviços ou ferramentas da Microsoft ou de terceiros por um período limitado. [Saiba mais](hololens2-options.md#learn-about-the-development-edition)

### <a name="can-any-edition-of-hololens-2-be-used-to-develop"></a>Qualquer edição do HoloLens 2 pode ser usada para desenvolver?

Sim. Qualquer edição do HoloLens 2 pode ser usada para desenvolver aplicativos.

### <a name="how-and-will-i-receive-my-500-in-azure-credits-3-month-unity-pro-trial-and-3-month-pixyz-plugin-trial"></a>Como e eu receberei meu $500 em créditos do Azure, avaliação do Unity pro de 3 meses e avaliação do plugin Pixyz de 3 meses?

Os créditos de desconto ou o acesso de avaliação serão enviados para o email usado na compra dentro de 2 dias úteis de remessa de produto.

### <a name="can-i-upgrade-my-hololens-2-device-only-device-to-the-hololens-2-development-edition-to-get-the-trial-credits"></a>Posso "atualizar" meu dispositivo de meu HoloLens 2 (somente dispositivo) para a edição de desenvolvimento do HoloLens 2 para obter os créditos de avaliação?

Não, não oferecemos planos de transferência de dispositivo. Não há capacidade de transferir de nenhuma edição para nenhuma outra edição.

### <a name="what-is-unity-pro"></a>O que é o Unity pro?

O Unity pro é uma plataforma 3D em tempo real para estúdios/empresas de pequeno/médio porte. Ele foi projetado para equipes com uma combinação de habilidades técnicas e não técnicas. [Saiba mais](https://store.unity.com/products/unity-pro)

### <a name="what-is-pixyz-plugin"></a>O que é o plug-in Pixyz?

O plug-in Pixyz lê e converte praticamente qualquer formato, otimiza-o e coloca-o no Unity e o prepara para fornecer ativos prontos para uso para aplicativos em tempo real. [Saiba mais](https://unity.com/products/pixyz)

### <a name="when-will-financing-for-the-hololens-2-development-edition-be-available"></a>Quando o financiamento da edição de desenvolvimento do HoloLens 2 estará disponível?

O financiamento para a edição de desenvolvimento do HoloLens 2 estará disponível no início de 2021.

### <a name="who-do-i-contact-with-questions-about-my-hololens-2-development-edition-entitlements"></a>Quem devo entrar em contato com perguntas sobre meus direitos de edição de desenvolvimento do HoloLens 2?

Com perguntas relacionadas a direitos, entre em contato com seu especialista em vendas às 866-425-4709, segunda-feira a Sex, 6h às 18:00, PST.

### <a name="can-my-company-buy-multiple-hololens-2-development-edition-devices"></a>Minha empresa pode comprar vários dispositivos do HoloLens 2 Development Edition? 

Este dispositivo destina-se a desenvolvedores que criam soluções de realidade misturada. Limitamos uma compra por transação de cliente. Se sua empresa quiser comprar vários dispositivos de edição de desenvolvimento, ele exigirá transações separadas para comprar dispositivos de edição de desenvolvimento do HoloLens 2 em nome da empresa. 

### <a name="can-i-show-my-hololens-2-development-edition-device-to-my-customers"></a>Posso mostrar meu dispositivo de edição de desenvolvimento do HoloLens 2 para meus clientes? 

Sim, este dispositivo destina-se a ser usado para protótipos e testes de soluções de realidade misturada.

### <a name="where-can-i-buy-the-hololens-2-development-edition"></a>Onde posso comprar a edição de desenvolvimento do HoloLens 2? 

O HoloLens 2 Development Edition é vendido exclusivamente em Microsoft Store.  

### <a name="can-i-return-my-hololens-2-development-edition"></a>Posso devolver meu HoloLens 2 Development Edition?  

Todas as vendas são finais e não há reembolsos disponíveis na edição de desenvolvimento do HoloLens 2.

### <a name="does-developer-edition-have-different-hardware-requirements-that-differ-from-the-hololens-2"></a>O Developer Edition tem requisitos de hardware diferentes que diferem do HoloLens 2?

Não, esse dispositivo é o mesmo de uma perspectiva de requisitos de hardware. A edição de desenvolvimento tem permissão para ser implantada no auxiliar do desenvolvimento e/ou teste de aplicativos em uma configuração comercial. 

### <a name="how-can-i-use-mixed-reality-toolkit--unity-for-developing-on-my-hololens-2-development-edition"></a>Como posso usar o kit de ferramentas de realidade misturada – Unity para desenvolvimento no meu HoloLens 2 Development Edition?

MRTK-Unity é um projeto conduzido pela Microsoft que fornece um conjunto de componentes e recursos usados para acelerar o desenvolvimento de aplicativos MR de plataforma cruzada no Unity. [Saiba mais](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### <a name="how-can-i-use-mixed-reality-toolkit--unreal-uxt-for-developing-on-my-hololens-2-development-edition"></a>Como posso usar o kit de ferramentas de realidade misturada – UXT inreal para o desenvolvimento no meu HoloLens 2 Development Edition?

MRTK-Unreal é o primeiro componente de ferramentas de UX a ser liberado e atualmente só tem suporte no HoloLens 2. O plug-in do componente inclui código, blueprints e ativos de exemplo de recursos de UX comuns para simulação de entrada, atores de interação à mão, componentes de botões que podem ser pressionados, componentes de manipulador e componentes de comportamento de acompanhamento. [Saiba mais](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

### <a name="are-os-releases-for-the-hololens-2-development-edition-the-same-as-hololens-2"></a>As versões do sistema operacional para o HoloLens 2 Development Edition são as mesmas do HoloLens 2?

Sim.

### <a name="can-i-run-released-sideload-or-store-applications-on-my-hololens-2-development-edition-device"></a>Posso executar aplicativos liberados (Sideload ou Store) no meu dispositivo de edição de desenvolvimento do HoloLens 2?

Se você estiver cumprirdo pelos termos e condições da edição de desenvolvimento do HoloLens 2, sim, você poderá executar qualquer aplicativo liberado na Developer Edition. Não limitamos o uso de Sideload ou aplicativos armazenados para esta SKU.

### <a name="where-can-i-learn-more-about-the-hololens-2-development-edition"></a>Onde posso saber mais sobre a edição de desenvolvimento do HoloLens 2?

Visite a página de documentação do HoloLens 2 Development Edition. [Saiba mais](https://docs.microsoft.com/hololens/hololens2-options?tabs=device)

### <a name="where-can-i-learn-more-about-developing-mixed-reality-solutions"></a>Onde posso saber mais sobre como desenvolver soluções de realidade misturadas?

Visite a documentação introdução ao desenvolvimento de realidade misturada. [Saiba mais](https://docs.microsoft.com/windows/mixed-reality/develop/development?tabs=unityhttps://docs.microsoft.com/en-us/windows/mixed-reality/develop/development?tabs=unity)

### <a name="where-can-i-learn-more-about-designing-mixed-reality-solutions"></a>Onde posso saber mais sobre como criar soluções de realidade misturada?

Visite a documentação introdução ao design de realidade misturada. [Saiba mais](https://docs.microsoft.com/windows/mixed-reality/design/design)

## <a name="azure"></a>Azure

### <a name="how-can-i-redeem-my-500-azure-credit"></a>Como posso resgatar meu crédito do Azure $500?

Para resgatar seus créditos do Azure $500, resgate seu Azure Pass código promocional e ative sua assinatura. [Saiba mais](https://www.microsoftazurepass.com/Home/HowTo?Length=5)

Examine o [contrato de assinatura](https://azure.microsoft.com/support/legal/subscription-agreement/?country=US) Azure Pass e os [detalhes da oferta](https://azure.microsoft.com/en-us/offers/azure-pass/).

### <a name="when-does-my-500-azure-credit-expire"></a>Quando meu crédito do Azure $500 expira?

Seu crédito do Azure $500 expirará em 90 dias após a ativação do código de chave do Azure.

### <a name="what-can-i-do-with-my-azure-credits"></a>O que posso fazer com meus créditos do Azure?

Com seus créditos do Azure, você pode começar a criar com o conjunto de serviços de nuvem em constante expansão da Microsoft&#39;s. Os créditos do Azure incluídos na edição de desenvolvimento do HoloLens 2 são para permitir que os clientes obtenham uma introdução rápida à criação e ao teste de aplicativos de realidade misturada no Azure. Para obter detalhes completos sobre a criação no Azure, visite [recursos do Azure](https://azure.microsoft.com/en-us/resources/)

### <a name="where-can-i-learn-more-about-azure-mixed-reality-services"></a>Onde posso saber mais sobre os serviços de realidade misturada do Azure?

A Microsoft oferece um conjunto de serviços de realidade misturada para permitir que os desenvolvedores criem e testem soluções de realidade mista. [Saiba mais](https://docs.microsoft.com/windows/mixed-reality/develop/mixed-reality-cloud-services)
